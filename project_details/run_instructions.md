# Portfolio_Dev Project Details

This is a modern React portfolio project built with Vite, Tailwind CSS, and Framer Motion.

## Prerequisites
- Node.js (version 16 or higher is recommended)
- npm (comes with Node.js)

## How to Run the Project

1. **Install Dependencies**
   First, you need to install all the required packages. Open your terminal, ensure you are in the root directory (`Portfolio_Dev`), and run:
   ```bash
   npm install
   ```

2. **Start Development Server**
   To start the local development server and work on the project, run:
   ```bash
   npm run dev
   ```
   This will start Vite on a local port (usually `http://localhost:5173`). You can open this URL in your browser to view the application.

3. **Build for Production**
   To build the project for production deployment, run:
   ```bash
   npm run build
   ```
   This creates an optimized, ready-to-deploy production build in the `dist` folder.

4. **Preview Production Build**
   If you want to preview the production build locally before deploying, you can run:
   ```bash
   npm run preview
   ```

## Tech Stack
- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (with `clsx` and `tailwind-merge` for class management)
- **Animations:** Framer Motion
- **Icons:** Lucide React
